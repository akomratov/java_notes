# Collections

## ArrayList

Syntax sugar:
```java
ArrayList<String> list = new ArrayList<String>();
list.add("Hello");
list.add("world");

ArrayList<String> list1 = new ArrayList<String>()
{{
   add("Hello");
   add("world");
}};

var list2 = new ArrayList<String>()
{{
   add("Hello");
   add("world");
}};
```
